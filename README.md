employees-scala-play-scalikejdbc
================================

Simple CRUD application with: 
- Scala 
- Play 2.4.x 
- ScalikeJDBC (ORM) (http://scalikejdbc.org/)
- AngularJS 
- json4s (http://json4s.org/)
- H2 

Usage
-----
- checkout code 
- run using activator 
```bash
$ activator run
```
- acccess <code>http://localhost:9000</code> url

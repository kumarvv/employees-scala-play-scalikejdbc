employees-scala-play-scalikejdbc
================================

Simple CRUD application with: 
- Scala 2.11.x
- Play 2.4.x 
- ScalikeJDBC (ORM)
- AngularJS 
- json4s 

Usage
-----
- checkout code 
- run using activator 
```bash
$ activator run
```
- acccess <code>http://localhost:9000</code> url
